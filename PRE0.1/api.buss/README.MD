# BUSSAPI CLI V1.0

A way to use the api.buss.lol API with the terminal

Features:

+ Fetch TLDs
+ Register a Domain
+ Delete a Domain

Usage:

```bash
    apibuss [domain] [--register (OPTIONAL)] [--delete <key> (OPTIONAL)] [--debug (OPTIONAL)]
```
